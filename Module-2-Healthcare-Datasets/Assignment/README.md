# Module 2 – Working with Mock Healthcare Datasets

## Purpose
I worked with a mock healthcare dataset of five patient clinic visits, organized the data in a spreadsheet, and calculated basic statistics like average patient age and average heart rate. The point was to get hands-on practice with real-world health data formats.

## Key Takeaways
Even with just five records, I could see how structured data makes it possible to spot patterns quickly. Calculating simple averages gave me a sense of how clinicians and AI systems alike rely on clean, well-organized data to draw meaningful conclusions.

## Insights
This exercise drove home something I keep coming back to throughout the course: AI tools are only as useful as the data they receive. If patient records are messy or incomplete, no algorithm will fix that. Getting comfortable with data handling felt like a necessary first step before exploring anything more complex.
