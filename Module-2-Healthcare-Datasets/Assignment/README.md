# Module 2 – Working with Mock Healthcare Datasets

## Purpose
This assignment involved analyzing a mock healthcare dataset of five patient clinic visits. The task was to organize the data in a spreadsheet and calculate basic statistics, including average patient age and average heart rate, to practice working with real-world health data formats.

## Key Takeaways
Even a small dataset reveals patterns that matter in healthcare. Calculating averages for age and heart rate across five patients demonstrated how structured data allows clinicians and AI systems to identify trends quickly. Understanding how to clean and organize healthcare data is a foundational skill for any AI application in this space.

## Insights
This exercise reinforced the idea that AI tools are only as useful as the data they receive. If patient records are inconsistent, incomplete, or poorly structured, even the most advanced algorithm will struggle to produce meaningful results. Getting comfortable with data handling is a necessary first step before exploring more complex AI-driven analytics.
