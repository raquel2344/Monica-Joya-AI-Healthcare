# Module 3 – Future Scenarios for AI-Powered Clinical Decision Support Systems

## Purpose
I was asked to envision where AI-powered clinical decision support systems (CDSS) could be in the next five to ten years. Using platforms like Glass Health as a starting point, I explored how CDSS might evolve from query-based tools into proactive, ambient clinical intelligence embedded directly within electronic health records.

## Key Takeaways
Right now, platforms like Glass Health still require clinicians to manually enter patient data and request recommendations. I believe the future lies in multimodal data fusion, where AI systems can analyze imaging, lab results, clinical notes, and genomic profiles in real time without waiting for a clinician to ask. But getting there requires building trust, preventing automation bias, and creating clear liability frameworks.

## Insights
The biggest thing I took away from this assignment was that technical capability alone is not enough. Even if these tools become highly accurate, widespread adoption will depend on transparency, training, and regulation. If clinicians do not trust the system, it does not matter how good the technology is.
