# Module 3 – Future Scenarios for AI-Powered Clinical Decision Support Systems

## Purpose
This assignment asked us to envision where AI-powered clinical decision support systems (CDSS) could be in the next five to ten years. Using platforms like Glass Health as a starting point, the paper explored how CDSS might evolve from query-based tools into proactive, ambient clinical intelligence embedded within electronic health records.

## Key Takeaways
Current CDSS platforms still require clinicians to manually enter patient data and request recommendations. The future of these tools lies in multimodal data fusion, where AI systems can simultaneously analyze imaging, lab results, clinical notes, and genomic profiles in real time. However, realizing this vision requires building clinician trust, preventing automation bias, and establishing clear liability frameworks.

## Insights
The biggest takeaway from this assignment was that technical capability alone is not enough. Even if AI-CDSS tools become highly accurate, widespread adoption will depend on transparency, training, and regulatory oversight. Without those, clinicians are unlikely to trust or effectively use these tools, regardless of how well they perform on paper.
